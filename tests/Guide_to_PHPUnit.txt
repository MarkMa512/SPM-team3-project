For macOS: 
1. Install Composer with the following command: 

$ curl -sS https://getcomposer.org/installer | php
$ sudo mv composer.phar /usr/local/bin/composer

2. To validate your installation, run the following command: 

$ composer --version

-- You should see something like this: 
"Composer version 2.1.9 2021-10-05 09:47:38"
-- You should also have a new sub-directory named "vendor" in your project directory, 
as well as a "composer.json" and a "composer.lock" file. 

3. Then, open your project directory in terminal, run the following commnad: 

$ composer require --dev phpunit/phpunit ^9

-- You should see something like this: 
"PHPUnit 9.5.10 by Sebastian Bergmann and contributors." 

4. Create a sub-directory called "tests" in your porking directory, 
put the class to be tested outside of the "tests" folder, 
while put the testClass into the "tests" folder. 

5. Run the following command: 

$ php vendor/bin/phpunit tests/testClass.php

-- You shoud see something like this: 
"PHPUnit 9.5.10 by Sebastian Bergmann and contributors.

...                                                                 3 / 3 (100%)

Time: 00:00.008, Memory: 4.00 MB

OK (3 tests, 3 assertions)" 

Happy Coding and testing!
